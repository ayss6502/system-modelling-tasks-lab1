# Task 1: ATM Cash Withdrawal System 
This module models the steps for withdrawing cash from an ATM machine, including: 
* PIN verification with 3-attempt limit. 
* Account balance query. 
* Withdrawal amount validation (multiples of 20TL). 
* Daily withdrawal limit checks.

# Task 2: Online Shopping Cart System
This system models the product browsing, cart management, and checkout processes of an e-commerce website.
Key features include:
* User login and stock availability checks.
* Cart editing capabilities.
* Minimum order limit ($10) and conditional free shipping (over $40).
* Payment method selection and order confirmation.

# Task 3: Hospital Appointment and Test Results System
This system models the patient experience in a digital hospital portal.
Key components:
* Patient identity verification via ID number.
* Appointment module: Clinic and doctor selection with a time slot selection loop.
* SMS notification upon successful booking.
* Test Results module: Multi-stage check for test existence and result readiness.
* PDF download feature for completed test results.

# Task 4: University Course Registration System
This system models the multi-layered logic required for student course enrollment.
Key features:
* Secure login with Student ID and Password.
* Course management loop for adding and dropping courses.
* Nested conditions: Capacity check, prerequisite validation, schedule conflict detection, and a 35-credit limit.
* Conditional advisor approval for students with a GPA below 2.5.
* Final registration summary and confirmation.

# Task 5: Smart Home Security System
This system models the alarm and notification processes of an IoT-based home security setup.
Key features include:
* Continuous sensor reading loop (Motion and Door/Window).
* Multi-level alarm logic (Low, Medium, High) based on detected activity.
* Homeowner presence check to prevent false alarms.
* Multi-channel notifications via SMS, App, and Email.
* Camera activation upon sensor triggering.
